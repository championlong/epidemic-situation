## 前言

`epidemic-situation`针对疫情环境下的便民超市等环境的排队问题进行研究。系统开发为解决减少人员聚集，防范公共场所人流量分布不均横的现象，有效缓解人口排队聚集节省时间效率。

## 项目介绍
> epidemic-situation是一个基于 vue 和 springboot 开发的全栈前后端分离的平台。主要为用户端的相关业务包括全国疫情实时统计展示、用户周边地点人员聚集情况、实时资讯展示栏目等模块开发。
